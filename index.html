空間設計大賽

空間設計大賽

活動目的

在都市創新快速進步的環境中，多功能複合式辦公室已然成為現在使用者的趨勢，如何在有限的空間內發揮最大的使用價值，為設計之目的及初衷，且促進設計的正向發展。只要是對空間創意與室內設計有想法、對社會有責任感的設計者，無論老將還是新秀，均可透過該獎項同台競技，鼓勵創意設計交流，發掘創意設計人才。

EVENT DESCRIPTION

評審標準

評審標準由豐邑機構總經理、中華民國室內設計協會理事長、台中科大副教授逢甲大學助理教授，組成評審團進行評選。

評審分為三個階段：

1.初審：將不符合評審內容之作品剔除（無規劃兩層樓，尺度大小不符合等..）

2.複審：選出8名較優秀參賽作品。

3.決賽：從這優秀作品中，參賽者來現場講解設計理念，由評審現場公佈最終評比結果。
